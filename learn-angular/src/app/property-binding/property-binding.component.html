6.Property Binding

It is used to change the change or update the property or attribute.

Same thing we can achieve through Interpolation, but interpolation doesn't work well with boolean value, work well only with string and number value.

Through Interpolation

Here, in Interpolation it is treating disable variable(in .ts file) value as a string not boolean type.

Through Property Binding

If you will change the value of disable variable(in .ts file) to true, input box will be disabled and vice versa.

Note: To change the property and attribute always use property binding.
